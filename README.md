# 📊 Sales Analytics Dashboard

## 📌 Project Overview

The **Sales Analytics Dashboard** is an interactive Power BI solution designed to analyze sales performance, monitor key business metrics, and provide actionable insights for decision-making. The dashboard transforms raw sales data into meaningful visual reports that help businesses understand revenue trends, customer behavior, and product performance.

---

## 🎯 Project Objective

The objective of this project is to:

* Monitor overall sales performance.
* Track revenue and profit trends.
* Analyze customer purchasing behavior.
* Identify top-performing products and regions.
* Support business decisions using data-driven insights.

---

## 🛠️ Tools & Technologies

* Microsoft Power BI
* Microsoft Excel
* Power Query
* DAX (Data Analysis Expressions)

---

## 📂 Dataset

The dataset contains sales transaction information, including:

* Order ID
* Order Date
* Customer Details
* Product Category
* Sales Amount
* Profit
* Quantity Sold
* Region

---

## 📈 Key Performance Indicators (KPIs)

* Total Sales
* Total Profit
* Total Orders
* Total Customers
* Average Order Value
* Profit Margin (%)

---

## 📊 Dashboard Features

* Executive KPI Cards
* Monthly Sales Trend Analysis
* Region-wise Sales Performance
* Product Category Analysis
* Customer Sales Distribution
* Profit Analysis
* Interactive Filters and Slicers

---

## 💡 Business Insights

* Identifies the highest-performing regions.
* Highlights best-selling product categories.
* Tracks monthly sales and profit trends.
* Helps identify business growth opportunities.
* Supports strategic decision-making through interactive visualizations.
